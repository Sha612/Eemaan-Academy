# Eemaan Academy Management System

A web-based Quran institute management system built with Next.js, React, Tailwind CSS, and role-based dashboard architecture.

## Features

- Admin dashboard
- Teacher dashboard
- Student dashboard
- Role-based navigation
- Student management
- Teacher management
- Class management
- Assign teacher to class
- Assign student to class
- Replacement teacher access
- Attendance overview
- Login UI and protected dashboard layout

## Tech Stack

- Next.js
- React
- TypeScript
- Tailwind CSS
- shadcn/ui
- PostgreSQL planned
- NestJS planned

## Project Status

Frontend dashboard in progress. Backend and database integration planned.

## Roles

- Admin

- Teacher
- Student
